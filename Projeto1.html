O que é o HTML?

HTML= HyperText Markup Language

Da maneira mais simplória possível, o HTML é uma "linguagem"Mais sobre isso depois que age como o esqueleto de uma página web, fornecendo a ela sua estrutura base. Logicamente, o mesmo vai além disso, também fornecendo funções que auxiliam na navegação da página, criação de listas ordenadas e não ordenadas, entre outras coisas. Comumente usada em blogs e qualquer página web num geral, ter um bom domínio de HTML é o primeiro passo para qualquer página ter estrutura.


Como funciona a navegação na Web?

Sabemos que o HTML dá estrutura para páginas web. Mas afinal, o que são e como funcionam?

Antes de mais nada, é importante esclarecer a diferença entre aplicações e páginas web. Uma página web é qualquer site que o usuário abre por meio de um navegador, enquanto uma aplicação é aberta por seu próprio "Ícone". Um exemplo é a Netflix, que ao ser aberta por um navegador é uma página web, mas ao ser aberta pelo "Ícone" de aplicação, se torna uma aplicação.


Em páginas web, tudo funciona com base em requisições entre dois tipos diferentes de servidores: O servidor de usuário, e o servidor host. O servidor usuário repreenta a máquina usada pelo usuário para acessar o site, seja seu celular, computador, tablet, ou o que quer que seja, enquanto o servidor host representa a máquina no qual a página web está hospedada. Toda vez que o usuário faz algo no site, como clicar em um link, expandir a página, ele faz uma requisição ao servidor host. Essa requisição é comunicada para o host, que retorna uma "resposta" apropriada para a mesma. Existem complexidades entre essa interação, como firewalls(Medidas de segurança que impedem requisições maliciosas e danosas de serem realizadas), e o papel de navegadores como interpretadores da página web, a possibilitando de ser aberta. Mas de forma simplificada, essa relação entre ambos os servidores é feita por meio de requisições e respostas do servidores entre si.

Tags do HTML

O que são, e o que fazem, exatamente?

Tags, da maneira mais simples possível, são as coisas que utilizamos em nosso código HTML para gerar sua estrutura e o que está contido no mesmo. Parágrafos, listas, textos, até mesmo deixar algo em negrito ou em itálico, tudo no HTML necessita de uma tag para existir e funcionar. Entre as mais notáveis e mais usadas, existem as tags de texto h /h e p /p, a tag strong que deixa o que está contido na mesma em negrito, a tag i que deixa em ,italico, entre várias outras. Também é importante mencionar a tag a, que consegue criar um link, que pode, por exemplo, te jogar para outras partes do texto, como o Começo deste pequeno texto. Num geral, ainda tenho muito mais para estudar e aprender, porém, já consigo ver algumas das muitas coisas que esta linguagem pode proporcionar.
